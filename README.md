# Voxel World Survival

一个基于 **HTML + Three.js** 的浏览器 3D 像素生存游戏，目标是做成“打开网页就能玩”的类《我的世界》体验。

仓库地址：<https://github.com/daved1210/test16>

## 项目特色

- 无需打包，核心版本可直接通过 `index.html` 启动
- 纯前端本地存档，使用 `localStorage` 保存世界与玩家进度
- 程序化体素地形生成
- 多种地表生态：平原、森林、沙漠、雪地、湿地、火山地带
- 生存玩法：挖掘、放置、背包、快捷栏、2x2 / 3x3 合成
- 实体系统：羊、猪、村民、僵尸、史莱姆
- 战斗反馈：即时伤害、受击闪红、击退、伤害飘字、头顶血条
- 建筑生成：房屋、村庄、围栏、农田、小院、灯柱
- 昼夜循环、雾效、贴图细节与基础光照氛围

## 在线 / 本地运行

### 方式一：直接打开

直接双击 `index.html` 即可运行。

> 说明：部分浏览器在 `file://` 模式下对交互、指针锁定、资源加载会更严格。
> 如果遇到异常，建议使用下面的本地服务器方式。

### 方式二：本地静态服务器（推荐）

在项目目录执行：

```bash
python -m http.server 8000
```

然后浏览器打开：

```text
http://127.0.0.1:8000/
```

## 操作说明

- `点击屏幕`：开始游戏 / 锁定鼠标
- `W A S D`：移动
- `空格`：跳跃
- `Shift`：下落
- `长按左键`：挖掘方块
- `左键点击实体`：攻击
- `右键`：放置方块 / 打开合成台
- `1 ~ 9`：切换快捷栏
- `E`：打开背包
- `Esc`：关闭当前面板

## 当前玩法内容

### 1. 世界与地形

- 区块化体素世界
- 程序化生成地形高度
- 不同生物群系自动切换表层材质
- 树木、岩浆、水域等自然内容
- 房屋与村庄会在合适区域自动生成

### 2. 生存与建造

- 可挖掘草方块、泥土、石头、木头、沙子等资源
- 可把资源放回世界进行建造
- 支持快捷栏和完整背包
- 支持 2x2 背包合成与 3x3 合成台合成

### 3. 战斗与实体

- 白天常见被动生物：羊、猪、村民
- 夜晚会出现敌对生物：僵尸、史莱姆
- 实体具有：
  - 生命值
  - 头顶血条
  - 即时伤害反馈
  - 受击闪红
  - 击退效果
  - 死亡掉落

### 4. HUD 与交互

- 准星目标信息
- 挖掘进度条
- 生命值显示
- 时间 / 模式显示
- 调试信息面板
- 消息提示

## 存档说明

游戏会自动保存以下内容到浏览器本地：

- 玩家位置与生命值
- 背包内容
- 合成栏状态
- 世界修改内容
- 时间进度
- 实体状态

如果你想重开，可以直接清除浏览器对应站点的 `localStorage`。
